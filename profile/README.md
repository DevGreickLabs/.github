### O que é

Lab independente de ferramentas de segurança ofensiva e defensiva, com foco em automação de threat intel, análise de IOCs e segurança de pipelines.

Mantido por [@DevGreick](https://github.com/DevGreick), Cybersecurity Analyst.

### Projetos ativos

#### [ThreatDeflect](https://github.com/DevGreickLabs/ThreatDeflect)
Engine de análise de ameaças. Automatiza consulta de IPs/URLs em múltiplas fontes (VirusTotal, Shodan, AbuseIPDB), correlaciona resultados e gera relatórios sumarizados com IA local via Ollama.
**Stack:** Python · Rust (PyO3) · `pip install threatdeflect-core`

#### [VulnHunter](https://github.com/DevGreickLabs/VulnHunter)
Scanner offline de CVEs em dependências de projeto. Roda sem chamar API externa, ideal para pipelines air-gapped.
**Stack:** Python · NVD feed local

#### [awesome-llm-security](https://github.com/DevGreickLabs/awesome-llm-security)
Curadoria de tools, frameworks e papers para segurança de LLMs, agentes e infraestrutura de IA.

#### [awesome-cyber-arsenal](https://github.com/DevGreickLabs/awesome-cyber-arsenal)
Curadoria multi-domínio: ofensiva, defensiva, CTI e DevSecOps.
